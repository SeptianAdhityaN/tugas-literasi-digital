Cara install Office 2021 menggunakan CMD secara GRATIS !!

Berikut adalah cara menginstall Office 2021 secara gratis dan mudah. silahkan ikuti step bye step nya ^^

Click Here

1. Buka Chrome lalu ketik "miscrosoft odt"

Klik yang Office "Deployment Tool Miscrosoft". Click Here !

Setelah masuk ke link nya, lalu klik tombol "Download" di bagian Office Deployment Tool.

2. Install software yang telah didownload

Masuk ke folder / tempat anda menyimpan file download, kemudian double klik.

Centang "Click here to accept the Miscrosoft Software License Terms" , lalu klik "Continue".

Kemudian buat folder baru untuk menyimpan file ekstraknya. klik "Make new folder", bebas diberi nama apa misalnya "Office Installation". Kemudian klik "Ok".

Jika sudah berhasil, akan muncul notifikasi "Files extracted successfully". Kemudian klik "Ok".

3. Edit Configuration

Masuk ke folder yang telah kita bikin sebelumnya "Office Installation"

Karena kita ingin menginstall Office 2021, maka kita delete 3 files di atas.

Untuk mengedit files configuration nya, klik files nya. kemudian klik kanan pada mouse lalu cari "Open with", klik "notepad".

Jika laptop kalian menggunakan 64 bit maka tidak perlu diubah, tetapi jika kalian menggunakan laptop 32 bit maka ubahlah 64 menjadi 32.

Kemudian ubah product ID seperti pada foto berikut. setelah itu, klik "Ctrl + S" lalu close tab.

4. Download Office menggunakan CMD

Klik "Shift + Klik kanan pada mouse" lalu pilih Open PowerShell.

kemudian ketik command ini untuk mendownload " .\setup.exe /download .\configuration-Office2021Enterprise.xml " lalu klik enter. PASTIKAN ANDA TERHUBUNG DENGAN INTERNET !!

Jika anda tidak dapat mengetik, berarti proses download sedang berjalan. File download sekitar 1-2 GB. silahkan ditunggu hingga semua proses selesai.

5. Install Office menggunakan CMD

Silahkan ketik command ini untuk menginstall Office 2021 pada kompuetr anda " .\setup.exe /configure .\configuration-Office2021Enterprise.xml "

kemudian akan muncul tampilan seperti ini, tunggu sebentar saja.

kemudian ini adalah proses instalasi nya, tunggu sampai prosesnya selesai.

Jika sudah selesai, maak anda akan mendapatkan notifikasi "Installation finished" yang berarti anda telah berhasil menginstall Office 2021 pada komputer anda.
